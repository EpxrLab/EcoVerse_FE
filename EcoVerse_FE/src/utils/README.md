# Utils Folder

This folder contains **utility functions and helper methods** used across the React Vite project.

Utilities are **pure, reusable, and framework-agnostic** functions.  
They should not depend on React components, hooks, or UI logic.

---

## Folder Structure Example

```text
utils/
├── formatDate.js
├── formatCurrency.js
├── validateEmail.js
├── debounce.js
├── storage.js
├── constants.js
├── generateId.js
└── README.md
```
