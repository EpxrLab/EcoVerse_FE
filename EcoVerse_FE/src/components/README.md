# Components Folder

This folder contains all **reusable UI components** used across the React Vite project.

Components in this folder should be **independent, reusable, and presentational**.  
They can be shared between multiple pages and should not be tied to a specific route.

---

## Folder Structure Example

```text
components/
├── Button.jsx
├── Input.jsx
├── Modal.jsx
├── Navbar.jsx
├── Footer.jsx
├── Card.jsx
├── Loading.jsx
├── EmptyState.jsx
└── README.md
```
